# FinCore Frontend

A React + TypeScript frontend demonstrating all FinCore banking platform features.

## Features

- **Dashboard** - Overview with account balances and FX rates
- **Accounts** - Open and view bank accounts with balance display
- **Payments** - Initiate payments with idempotency, track status
- **FX Rates** - Live exchange rates, conversion quotes
- **Fraud Cases** - Compliance dashboard for fraud review (admin)
- **Notifications** - Channel routing visualization (email/push/SMS)

## Prerequisites

1. Start infrastructure: `cd infrastructure && docker compose up -d`
2. Build all backend: `mvn clean install -DskipTests`
3. Start services locally with `-Dspring.profiles.active=local`

## Running

```bash
cd frontend
npm install
npm run dev
```

Open http://localhost:5173

Login with Keycloak: `admin` / `admin` (or create users at http://localhost:8180)

Frontend connects to API Gateway at http://localhost:8080

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).
